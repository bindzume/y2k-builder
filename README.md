# Cloud 9 aka y2k-builder
This is a (heavily vibecoded) webapp that helps you build simple ugly websites without needing to master HTML or CSS

Try it here: https://y2k-builder-742966517145.us-central1.run.app/

## Features:
1. Project selector
   - Each html page is a project. With the project selector, you can work on multiple pages at once
2. Completely local backups of your project
   - All your project details are stored in your browser's storage. Autosave means you can close and come back anytime without worrying about losing your data
3. Backup and Load
   - You can backup your entire project and load it on another session using Builder Files
4. Export single pages or all of your projects
   - Ready for drag-and-drop into neocities
5. Lots of cool effects for all of your different elements
6. Flex boxes for formatting
